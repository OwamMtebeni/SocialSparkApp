# Social Spark App

Description
The Social Spark App is a simple Android application developed in Kotlin. It helps users maintain social connections by suggesting small social actions ("sparks") based on the time of day.

 Include these features:
- User input for time of day
- Displays social suggestions
- Reset button to clear input and output
- Error handling for invalid input

 Technology Used
- Kotlin
- Android Studio

 How to Use it
1. Enter a time of day (e.g. Morning, Afternoon, Dinner)
2. Click "Get Suggestion"
3. View the suggested social action
4. Click "Reset" to start again

 ⚠️ Error Handling
- Displays a message if input is empty
- Handles invalid time inputs with helpful feedback

 Setup Instructions
1. Clone the repository
2. Open in Android Studio
3. Run the app on an emulator or device
